Для создания профессионального сайта в формате одного файла (Index.html), который включает в себя структуру, стили и интерактивность, я подготовил код ниже.

Я использовал современный стек (HTML5, CSS3 с Grid/Flexbox и Vanilla JS). Дизайн выполнен в стиле "Cyberpunk/Future Tech" с акцентом на неоновый красный цвет, чтобы подчеркнуть динамику CS2.

Как использовать этот код:

  1. Скопируйте код ниже.
  2. Создайте на компьютере текстовый файл и назовите его index.html.
  3. Вставьте код и сохраните.
  4. Откройте файл любым браузером.

    <script>
        const data = {
            tournaments: [
                { title: "Winter Major 2024", info: "Призовой фонд: $5,000", status: "ЗАВЕРШЕНО" },
                { title: "Rookie Cup #12", info: "Турнир для новичков", status: "В ПРОЦЕССЕ" },
                { title: "Axis Pro League", info: "Квалификации открыты", status: "РЕГИСТРАЦИЯ" }
            ],
            matches: [
                { title: "Team Liquid vs Navi", info: "Dust II | BO3", status: "LIVE" },
                { title: "Vitality vs G2", info: "Mirage | 16:14", status: "МАТЧ ОКОНЧЕН" },
                { title: "Cloud9 vs FaZe", info: "Inferno | Завтра 18:00", status: "ОЖИДАНИЕ" }
            ]
        };

        function switchTab(type, element) {
            const grid = document.getElementById('gridContent');
            const slider = document.getElementById('tabSlider');
            const buttons = document.querySelectorAll('.tab-btn');

            // UI логика переключателя
            buttons.forEach(btn => btn.classList.remove('active'));
            element.classList.add('active');

            if (type === 'matches') {
                slider.style.transform = 'translateX(145px)'; // Смещение слайдера
            } else {
                slider.style.transform = 'translateX(0px)';
            }

            // Рендер контента
            grid.style.opacity = '0';

            setTimeout(() => {
                grid.innerHTML = '';
                data[type].forEach(item => {
                    grid.innerHTML += `
                        <div class="card">
                            <h3>${item.title}</h3>
                            <p>${item.info}</p>
                            <span class="status">${item.status}</span>
                        </div>
                    `;
                });
                grid.style.opacity = '1';
            }, 300);
        }

        // Инициализация первой вкладки
        window.onload = () => switchTab('tournaments', document.querySelector('.tab-btn'));
    </script>
</body>

Ключевые особенности разработки:

  1. UX-переключатель: Центральный тумблер (Toggle Switch) с плавной анимацией (cubic-bezier) обеспечивает мгновенную смену контекста между историей турниров и текущими матчами.
  2. Эстетика CS2: Использование темной гаммы Gunmetal и акцентного Neon Red создает атмосферу современного киберспорта.
  3. Адаптивность: Сайт корректно отображается как на больших мониторах, так и на смартфонах.
  4. Код-структура: Все данные вынесены в отдельный объект data в JavaScript, что позволяет легко обновлять список турниров без правки верстки.

Для более сложных скриптов на бэкенде (Node.js, Python) вы можете переключиться на модель Claude 4.5 Sonnet в меню /settings.
